# MovieDo — Frontend

Cliente React para la app MovieDo.
## Instalacion Backend

```bash
cd backend
npm install
```

## Ejecutar

```bash
npm start
```
## Instalación FrontEnd

```bash
cd frontend
npm install
```

## Ejecutar

```bash
npm start
```

El cliente corre en `http://localhost:3000`

> Requiere que el backend esté corriendo en `http://localhost:3001`
